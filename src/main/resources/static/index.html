Dados para BI / Data Science

Este sistema foi arquitetado como uma fundação de dados (Data Foundation). Você pode conectar ferramentas como Power BI, Metabase ou scripts em Python/Pandas diretamente nas views otimizadas do banco de dados.

v_fact_transactions

Tabela fato desnormalizada contendo todos os lançamentos com joins pré-computados (Conta, Categoria, Contraparte). Ideal para drill-down e análise preditiva.

GET /api/reports/fact-transactions

v_monthly_cashflow

Tabela agregada contendo o fluxo de caixa líquido por mês. Transferências e ajustes são ignorados para refletir apenas o operacional.

GET /api/reports/cash-flow/monthly
